# frontend_builder
FrontEnd builder [npm, gulp, bower]


Step 1: npm i
Step 2: bower i
Step 3: npm start

NPM :

  "start": запуск среды разработки
  "build": выгрузка для продакшина
  "deploy": выгрузка на GitHub-Pages


GULP :
  - less
  - browserSync
  - notify
  - postcss
  - gh-pages

+ ESlint
+ StyleLint
